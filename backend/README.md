# NetSim Backend

Real-time collaboration platform with Socket.IO and Node.js.

## Features

- ✅ Virtual room creation with cryptographic IDs
- ✅ Multi-user real-time chat
- ✅ Role-based access control (Host/Admin/Member)
- ✅ Private peer-to-peer channels
- ✅ Network topology visualization
- ✅ Message persistence with audit logs
- ✅ Automatic room lifecycle management
